css연습기록


### 2022/02/06 
- box-sizing : border-box 
  - 테두리를 포함한 크기를 width, height로 지정
- flex-direction : 축 결정 
  - row : 가로축
  - column : 세로축
- flex-wrap 
  부모요소의 넓이보다 자식요소의 넓이가 넓다면 
  - wrap:  줄바꿈  
  - nowrap : 자식요소의 넓이를 부모요소에 맞게 강제 축소 
  - wrap-reverse : 줄바꿈할때 역순으로 배열
  - inherit : 부모요소의 설정값 사용
